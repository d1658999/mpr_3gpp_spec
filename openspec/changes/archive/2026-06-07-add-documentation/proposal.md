## Why

Currently, there is no project documentation describing how to use the calculator, how calculations are performed, and what operating systems/browsers are supported. Users and engineers need a quick onboarding guide and reference manual to understand the interface and specification requirements without looking at code.

## What Changes

- **Add** `README.md` at the repository root outlining the tool overview, technology stack, features, installation, and deployment instructions.
- **Add** `GUIDE.md` at the repository root containing a step-by-step user manual with screenshots or text diagrams showing how to configure settings, interpret pass/fail outcomes, and verify note tolerances.
